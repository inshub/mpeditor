Make writing, formatting, and preparing WeChat Official Account articles simpler.

### Core Features

- Write more smoothly
- Format with less effort
- Publish more efficiently

You can think of it as a more convenient writing workspace:

- Write articles in one place
- See the result as you write
- Switch article styles quickly
- Upload images directly
- Copy content in a format that fits WeChat better
- Send drafts directly to your WeChat Official Account

It is designed for ordinary users. You do not need to know code or learn complicated formatting rules in advance.

## Before You Start

Some features need your own account settings, for example:

- WeChat draft publishing
- Image hosting
- AI cover generation ([Get API Key](https://modelscope.cn/my/access/token))

### WeChat Draft Publishing

**WeChat Forwarding API**: When using the WeChat forwarding API, you need to add the [Cloudflare API IP ranges](https://www.cloudflare.com/ips/) to the IP whitelist in WeChat Official Account Security Center.
